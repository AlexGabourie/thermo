thermo
==============

| 1. **gpumd** : Python interface specific to `GPUMD`_. 
| 2. **lammps** : Python interface specific to `LAMMPS`_.
| 3. **shared** : Used strictly to compare `GPUMD`_ and `LAMMPS`_.
